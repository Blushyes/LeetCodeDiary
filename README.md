# 总结

## 栈

### 关于单调栈

+ **单调栈的介绍**

> ​	**单调栈（monotone-stack）是指栈内元素（栈底到栈顶）都是（严格）单调递增或者单调递减的。**

+ **什么时候用**

**当需要找左边或右边第一个比自己大/小的位置用单调栈。**

例如[每日温度](./notes/medium/每日温度.md)中寻找右边比当前温度更高的温度。
